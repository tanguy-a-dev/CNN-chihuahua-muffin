# AI Chihuahua Muffin

A small PyTorch training exercise: learn to tell a chihuahua from a muffin with a simple CNN (`SnifferCNN`).

Everything lives in `notebook.ipynb`, load images from `data/train` and `data/test`, train for a few epochs, then run predictions on held-out and extra images.

Dataset used: [Muffin vs chihuahua](https://www.kaggle.com/datasets/samuelcortinhas/muffin-vs-chihuahua-image-classification/data)

Not production code, just practice with datasets, DataLoaders, and a minimal image classifier.
